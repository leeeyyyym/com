    Hello, C! — 개발 환경 준비와 첫 프로그램¶

    🎯 이 장에서 배우는 것¶

    • C언어 개발 환경(온라인 IDE)을 열고 코드를 실행할 수 있다
    • printf를 사용하여 화면에 원하는 문장을 출력할 수 있다
    • C 프로그램의 기본 구조 4요소(#include, main, {}, ;)를 설명할 수 있다

    📚 핵심 개념¶

    개념: C 프로그램의 기본 구조¶

    비유로 시작 ✉️

    C 프로그램은 공식 편지와 같아요. 친구에게 카톡 보낼 땐 "ㅎㅇ"면 되지만, 공식 편지에는 보내는 사람, 받는 사람, 본문, 마침표 같은 정해진 형식이 필요하죠. 파이썬이 카톡이라면, C는 공식 편지입니다.

    정확한 정의

    #include <stdio.h>    // 📮 "출력 도구를 가져와줘"

int main() {          // 📝 "여기서부터 편지 본문 시작"
    printf("Hello!");  // 💬 화면에 글자를 출력
    return 0;          // 👋 "편지 끝, 잘 마쳤어"
}
    • #include <stdio.h> → 출력(printf) 기능이 담긴 도구 상자를 여는 것
    • int main() → 프로그램이 시작되는 출발점
    • { } → 본문의 시작과 끝을 감싸는 울타리
    • ; → 문장 하나가 끝났다는 마침표 (빼먹으면 에러! 🚨)

    파이썬과 비교하면?

    flowchart LR A["🐍 파이썬\nprint('Hello!')"] -- "이게 전부!" --> B["화면 출력"] C["⚙️ C언어\n#include → main → printf → ;"] -- "형식을 갖춰야!" --> B

    파이썬은 한 줄이면 되지만, C는 4가지 형식을 모두 갖춰야 합니다. 번거롭지만, 이 덕분에 C는 컴퓨터가 아주 빠르게 실행할 수 있어요. 💨

    Step 2: 첫 프로그램 — MBTI 게임 타이틀 출력¶

    코드:

    #include <stdio.h>

int main() {
    printf("=============================\n");
    printf("  나의 MBTI를 알아보자!\n");
    printf("=============================\n");
    return 0;
}

    초록색 ▶ Run 버튼을 클릭하세요!

    실행 결과:

    =============================
  나의 MBTI를 알아보자!
=============================

    🎉 축하합니다! 첫 C 프로그램을 실행했어요!

    💡 \n은 줄바꿈 기호입니다. 파이썬의 print()는 자동 줄바꿈이지만, C의 printf()는 \n을 직접 넣어야 해요.

    📝 전체 코드¶

    #include <stdio.h>

int main() {
    printf("=============================\n");
    printf("  나의 MBTI를 알아보자!\n");
    printf("=============================\n");
    return 0;
}

    ⚠️ 주의할 점¶

    • 세미콜론(;) 빠뜨리기 → C에서 가장 흔한 실수 1위! 모든 문장 끝에 ;을 꼭 붙이세요.
    • #include 오타 → <stdio.h>에서 철자가 하나라도 틀리면 컴파일 에러가 납니다. 그대로 따라 적어주세요.

    ✅ 점검하기¶

    1. C 프로그램에서 #include <stdio.h>의 역할은 무엇인가요?

    정답 확인 `printf` 같은 입출력 함수를 사용하기 위해 **표준 입출력 도구 상자(라이브러리)**를 가져오는 것입니다.

    2. 아래 코드에서 에러가 나는 이유는?

    printf("Hello")
    정답 확인 문장 끝에 **세미콜론(`;`)**이 빠져 있습니다. `printf("Hello");`로 고쳐야 합니다.

    3. printf에서 줄바꿈을 하려면 어떤 기호를 쓰나요?

    정답 확인 `\n`을 문자열 안에 넣으면 줄바꿈됩니다. 예: `printf("첫줄\n둘째줄\n");`
